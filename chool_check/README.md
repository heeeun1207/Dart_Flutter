## 레이아웃 구상하기
- ‘오늘도 출첵’ 프로젝트 3분할 화면으로 레이아웃을 구성한다.

1. AppBar : 중앙에 타이틀 위치
2. Body : 구글 지도
3. Footer : 출근하기 기능을 구현할 영역
    1. 위치가 출근 체크가 가능한 영역이면 [출근하기] 버튼이 보인다.
    2. 범위가 아닐 경우 버튼이 보이지 않게 구현
