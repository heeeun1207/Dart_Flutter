# video_call
아고라 API를 이용해서 간단한 1:1 영상 통화 앱 만들기
- 실제 핸드폰을 이용해서 화상 통화 해보기
- 입장 & 종료 기능 만들기

## 핵심 요소

- 아고라 API
- 권한 관리
- WebRTC

## 사전 지식

- 카메라 플러그인
- WebRTC
- 에뮬레이터에서 카메라 사용
- 내비게이션

## 사전 준비

- 아고라 API
- 이미지, 폰트 추가
- pubspec.yaml 설정
- 네이티브 설정
- 플러터에서 권한 관리
- 프로젝트 초기화
