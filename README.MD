### 제네릭 문자

- T : 변수 타입을 사용할 때 흔히 사용된다. 예) T value;
- E : 리스트 내부 요소들의 타입을 표현할 때 흔히 사용된다. 예) List<E>
- K : 키를 표현할 때 흔히 사용된다. 예) Map<K,V>
- V : 값을 표현할 때 흔히 사용된다. 예) Map<K,V>
